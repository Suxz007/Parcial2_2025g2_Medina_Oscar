Sobre plataforma Educativa

1. Visión General

Aplicación de ejemplo para enseñar los fundamentos de una plataforma de aprendizaje en línea con:

  1. Interfaz de Usuario (Angular)
  2. Servidor y Lógica de Negocio (NestJS)
  3. Almacenamiento de Datos (PostgreSQL)
  4. Videoconferencias en Tiempo Real (WebRTC)
2. Interfaz de Usuario: Angular

¿Qué es? Framework de Google para SPAs.

Conceptos clave:

  1. Componentes reutilizables
  2. Data Binding bidireccional
  3. Angular CLI para scaffolding

Ejemplo didáctico:

// login.component.ts
export class LoginComponent {
  form = this.fb.group({
    email: [''],
    password: ['']
  });
  constructor(private fb: FormBuilder, private auth: AuthService) {}
  submit() {
    this.auth.login(this.form.value).subscribe();
  }
}
3. Servidor y Lógica de Negocio: NestJS

¿Qué es? Framework Node.js con TypeScript.

Conceptos clave:

  1. Controladores y Servicios
  2. Decorators para rutas e inyección
  3. WebSockets integrados

Ejemplo didáctico:

// auth.controller.ts
@Controller('auth')
export class AuthController {
  constructor(private authService: AuthService) {}
  @Post('login')
  login(@Body() dto: LoginDto) {
    return this.authService.login(dto);
  }
}
4. Almacenamiento de Datos: PostgreSQL

¿Qué es? Base de datos relacional avanzada.

Conceptos clave:

  1. Transacciones ACID
  2. JSONB para datos semiestructurados
  3. Índices para optimización

Ejemplo didáctico:

-- schema.sql
CREATE TABLE users (
  id SERIAL PRIMARY KEY,
  email VARCHAR(255) UNIQUE NOT NULL,
  password VARCHAR(255) NOT NULL
);
CREATE TABLE progress (
  id SERIAL PRIMARY KEY,
  user_id INT REFERENCES users(id),
  completed_quizzes INT DEFAULT 0
);
5. Videoconferencias en Tiempo Real: WebRTC

¿Qué es? API de navegador para P2P de audio y vídeo.

Conceptos clave:

  1. RTCPeerConnection para ofertas/respuestas
  2. Señalización vía WebSocket
  3. Streams multimedia sin plugins

Ejemplo didáctico:

// video.service.ts (Angular)
const pc = new RTCPeerConnection();
navigator.mediaDevices.getUserMedia({ video: true, audio: true })
  .then(stream => {
    pc.addStream(stream);
    // intercambiar señales con servidor...
  });
6. Flujo de Trabajo Sugerido
  1. Setup: ng new edulearn-ui, nest new edulearn-api, inicializar PostgreSQL.
  2. Autenticación: Implementar login/registro con JWT.
  3. Persistencia: Mapear entidades con TypeORM o Prisma.
  4. Videoconferencia: Configurar WebSocket y RTCPeerConnection.
  5. Pruebas y Documentación: README y comentarios en código.

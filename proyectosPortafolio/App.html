App Fintech

1. Visión General

Aplicación móvil de ejemplo para enseñar los fundamentos de una app fintech con:

  1. Gestión de Presupuestos Personales
  2. Seguimiento de Gastos en Tiempo Real
  3. Simulador de Inversiones
  4. Análisis Financiero con Gráficos
2. Desarrollo Móvil: React Native

¿Qué es? Framework de Facebook para desarrollo multiplataforma (iOS/Android).

Conceptos clave:

  1. Componentes nativos: View, Text, TouchableOpacity, ScrollView
  2. Navegación con React Navigation entre pantallas
  3. Hooks para estado local y efectos secundarios

Ejemplo didáctico:

// ExpenseScreen.js
import React, { useState } from 'react';
import { View, Text, TextInput, TouchableOpacity } from 'react-native';

export default function ExpenseScreen() {
  const [amount, setAmount] = useState('');
  const [category, setCategory] = useState('');
  
  const addExpense = () => {
    // Guardar gasto en Firebase
    firestore().collection('expenses').add({
      amount: parseFloat(amount),
      category,
      date: new Date()
    });
  };
  
  return (
    <View>
      <TextInput 
        value={amount} 
        onChangeText={setAmount} 
        placeholder="Monto" 
      />
      <TouchableOpacity onPress={addExpense}>
        <Text>Agregar Gasto</Text>
      </TouchableOpacity>
    </View>
  );
}
3. Backend y Base de Datos: Firebase

¿Qué es? Plataforma de Google para desarrollo de apps sin servidor.

Conceptos clave:

  1. Firestore: Base de datos NoSQL en tiempo real
  2. Authentication: Login con Google, Facebook o email/password
  3. Cloud Functions: Lógica de servidor para cálculos complejos

Ejemplo didáctico:

// firebase.js
import firestore from '@react-native-firebase/firestore';

export const getExpenses = (userId) => {
  return firestore()
    .collection('expenses')
    .where('userId', '==', userId)
    .orderBy('date', 'desc')
    .get();
};

export const getBudgetStatus = async (userId) => {
  const expenses = await getExpenses(userId);
  const total = expenses.docs.reduce((sum, doc) => 
    sum + doc.data().amount, 0
  );
  return { spent: total, remaining: 1000 - total };
};
4. Pagos y Transacciones: Stripe

¿Qué es? API de pagos para procesar transacciones seguras.

Conceptos clave:

  1. Payment Intents para flujo de pago seguro
  2. Webhooks para confirmar transacciones
  3. Stripe Elements para formularios de tarjetas

Ejemplo didáctico:

// payment.js
import { StripeProvider, CardField, confirmPayment } from '@stripe/stripe-react-native';

    if (paymentIntent.status === 'Succeeded') {

        amount: 100,
        type: 'deposit',

      });
    }
  };
}
5. Visualización de Datos: Chart.js

¿Qué es? Librería JavaScript para crear gráficos interactivos.

Conceptos clave:

  1. Gráficos de barras para gastos por categoría
  2. Gráficos de línea para evolución de ahorros
  3. Gráficos de dona para distribución del presupuesto

Ejemplo didáctico:

// ChartScreen.js
import { LineChart } from 'react-native-chart-kit';

export default function AnalyticsScreen() {
  const [chartData, setChartData] = useState({
    labels: ['Ene', 'Feb', 'Mar', 'Abr'],
    datasets: [{
      data: [500, 750, 600, 900],
      label: 'Gastos Mensuales'
    }]
  });

  return (
    <LineChart
      data={chartData}
      width={350}
      height={220}
      chartConfig={{
        backgroundColor: '#e26a00',
        backgroundGradientFrom: '#fb8c00',
        backgroundGradientTo: '#ffa726',
        color: (opacity = 1) => `rgba(255, 255, 255, ${opacity})`,
      }}
    />
  );
}
6. Flujo de Trabajo Sugerido
  1. Setup: npx react-native init FinApp, configurar Firebase project y Stripe account.
  2. Autenticación: Implementar login con Firebase Auth.
  3. CRUD Gastos: Crear, listar y categorizar gastos con Firestore.
  4. Presupuestos: Definir límites y notificaciones.
  5. Inversiones: Integrar Stripe para depósitos simulados.
  6. Analytics: Implementar gráficos con Chart.js para análisis.
  7. Testing: Probar en dispositivos iOS/Android y documentar.
7. Resultado Esperado

Al completar este proyecto, los estudiantes habrán aprendido a:

  1. Desarrollar apps móviles multiplataforma con React Native.
  2. Implementar autenticación y base de datos NoSQL con Firebase.
  3. Integrar pagos seguros usando Stripe API.
  4. Crear visualizaciones financieras con Chart.js.
  5. Manejar estado global y navegación en apps móviles.
